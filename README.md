# Saki
Saki is a blog generator with microCMS.

## Usage
In order to run this project, you need to install Node.js with pnpm package manager.
### Setup
```bash
pnpm i
```
And then create `.env` file.
```env
MICROCMS_SERVICE_DOMAIN=
MICROCMS_API_KEY=
```
### Development
```bash
pnpm dev
```
### Build
```bash
pnpm build
```
